# AI Eco Footprint

An interactive, single-page environmental footprint estimator with a focus on how AI usage fits into broader personal-impact categories like home energy, travel, food, waste, digital life, and device replacement.

## GitHub Pages

This is a static site. To publish it with GitHub Pages:

1. Open the repository on GitHub.
2. Go to Settings > Pages.
3. Set the source to Deploy from a branch.
4. Choose the `main` branch and `/ (root)`.
5. Save.

The site should publish from `index.html`.
